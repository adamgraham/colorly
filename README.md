# Colorly

> A versatile collection of color tools for designers and developers encapsulated in one web app.

### Included Tools

- Tints, Shades, Tones
- Color Harmonies
- Color Blindness Visualizer
- Contrast Ratio Compliance
- Hue Palette
- Saturation Palette
- Lightness Palette
- Gradient Preview

#### Other Features

- Copy to clipboard
- Undo/redo
- Randomize

#### Color Info

- HEX
- RGB
- CMYK
- HSL
- HSV
